PROJECT TITLE: Flying Fish!
PURPOSE OF PROJECT: Testing fullscreen mode
VERSION and DATE: 1.0 - 2019-02-27
HOW TO START THIS PROJECT: run project.greenfoot with Greenfoot 3.1.0(!) or run standalone jar
AUTHORS: Ferhat
USER INSTRUCTIONS:

- In main menu only: press F to run project in fullscreen mode.
- Use arrow keys to control your actor and press space key to shoot.
- infinite game without end unless player lost all health points.

REQUIREMENTS:
- Greenfoot 3.1.0
- Java JRE 8
